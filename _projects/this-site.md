The very site you're reading. Built with Jekyll and hosted on GitHub Pages.

## Goals

- Fast, static, no JS bloat
- Dark but not depressing
- Easy to update from a text editor
- A home for both professional work and weird side projects

## Technical notes

The theme is entirely custom SCSS — no frameworks, no Bootstrap, just variables and
nested rules. The starfield background is pure CSS using `radial-gradient` scattered
across the `body::before` pseudo-element.

Syntax highlighting is provided by Rouge with custom color overrides to match the
purple/teal palette.
